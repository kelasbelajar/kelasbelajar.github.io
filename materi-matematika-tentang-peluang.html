Jawab:

20.160

Penjelasan dengan langkah-langkah:

Pertama, kita hitung jumlah huruf dari 'Mouri Ran', yaitu 5 + 3 = 8 huruf

Kemudian, kita hitung jumlah masing-masing huruf:

M = 1

O = 1

U = 1

R = 2

I = 1

A = 1

N = 1

Maka, banyaknya permutasi adalah:

[tex]permutasi:\frac{8!}{1!\,1!\,1!\,2!\,1!\,1!\,1!} = \frac{40320}{2} = 20160[/tex]

Maka, terdapat 20.160 permutasi

Jawab:

Penjelasan dengan langkah-langkah:

Untuk menghitung peluang dari suatu kejadian, kita bisa menggunakan rumus peluang:

Peluang = (Jumlah kejadian yang diinginkan) / (Jumlah semua kejadian)

Diketahui:

Peluang A hidup 20 tahun lagi = 0,2

Peluang B hidup 20 tahun lagi = 0,4

a. Peluang A dan B akan hidup 20 tahun lagi:

Peluang (A dan B) = Peluang A x Peluang B = 0,2 x 0,4 = 0,08

b. Peluang A hidup 20 tahun lagi, tetapi B tidak:

Peluang (A hidup, B tidak) = Peluang A x (1 - Peluang B) = 0,2 x (1 - 0,4) = 0,2 x 0,6 = 0,12

c. Peluang B hidup 20 tahun lagi, tetapi A tidak:

Peluang (B hidup, A tidak) = Peluang B x (1 - Peluang A) = 0,4 x (1 - 0,2) = 0,4 x 0,8 = 0,32

d. Peluang A dan B keduanya tidak hidup 20 tahun lagi:

Peluang (A tidak hidup, B tidak hidup) = (1 - Peluang A) x (1 - Peluang B) = (1 - 0,2) x (1 - 0,4) = 0,8 x 0,6 = 0,48

Jadi, jawabannya adalah:

a. Peluang A dan B akan hidup 20 tahun lagi = 0,08

b. Peluang A hidup 20 tahun lagi, tetapi B tidak = 0,12

c. Peluang B hidup 20 tahun lagi, tetapi A tidak = 0,32

d. Peluang A dan B keduanya tidak hidup 20 tahun lagi = 0,48

Ketika kita berbicara tentang peluang dua peristiwa terjadi bersama-sama, kita mengalikan peluang masing-masing peristiwa. Mari kita selesaikan setiap pertanyaan:

a. Peluang bahwa A dan B akan hidup 20 tahun lagi adalah produk dari peluang A hidup dan peluang B hidup:

  \(P(A \text{ dan } B) = P(A) \times P(B) = 0.2 \times 0.4 = 0.08\)

b. Peluang bahwa A hidup 20 tahun lagi, tetapi B tidak hidup adalah peluang A hidup dikurangi peluang A dan B keduanya hidup:

  \(P(A \text{ dan } \neg B) = P(A) - P(A \text{ dan } B) = 0.2 - 0.08 = 0.12\)

c. Peluang bahwa B hidup 20 tahun lagi, tetapi A tidak hidup adalah peluang B hidup dikurangi peluang A dan B keduanya hidup:

  \(P(\neg A \text{ dan } B) = P(B) - P(A \text{ dan } B) = 0.4 - 0.08 = 0.32\)

d. Peluang bahwa A dan B keduanya tidak hidup 20 tahun lagi adalah komplementer dari peluang A atau B hidup:

  \(P(\neg A \text{ dan } \neg B) = 1 - (P(A) + P(B) - P(A \text{ dan } B)) = 1 - (0.2 + 0.4 - 0.08) = 0.48\)

Jadi, jawabannya adalah:

a. Peluang \(0.08\)

b. Peluang \(0.12\)

c. Peluang \(0.32\)

d. Peluang \(0.48\)Jawab:

Dalam satu kartu bridge ada sebanyak 52 kartu, jadi banyaknya semua titik sampel = 52.

Dalam satu kartu bridge terdiri dari 4 gambar yaitu,  Heart, diamond, spade, dan club .

b. Misalkan kejadian terambil satu kartu berwarna merah = B

Banyaknya kartu warna merah,  = 2 × 13 = 26, n(B)= 26

Banyaknya kartu seluruhnya ada 52, n(S) = 52.

Jadi, P(B) = n(B)/n(S) = 26/52 = 1/2

Karena soal dikerjakan dengan menebak, maka peluang jawab benar adalah 1/5 (20%) dan peluang jawab salah adalah 4/5 (80%).

Untuk menjawab 2 soal benar, kita bisa menggunakan rumus binomial:

C(6,2) x (1/5)^2 x (4/5)^4 = 15 x 0.04 x 0.32768 = 0.24

Untuk menjawab 3 soal benar, kita bisa menggunakan rumus binomial:

C(6,3) x (1/5)^3 x (4/5)^3 = 20 x 0.008 x 0.4096 = 0.164

Untuk menjawab 4 soal benar, kita bisa menggunakan rumus binomial:

C(6,4) x (1/5)^4 x (4/5)^2 = 15 x 0.0016 x 0.16 = 0.0448

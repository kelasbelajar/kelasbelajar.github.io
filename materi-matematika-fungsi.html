3. Matematika Materi Fungsi Komposisi​


Jawaban:

Diketahui fungsi f : R → R, g : R → R dirumuskan dengan f(x) = 2x - 1 dan g(x) = (x + 3) / (2 - x), x ≠ 2. Fungsi Invers dari f o g(x) = ....

A. (2x + 4) / (x + 3)

B. (2x - 4) / (x + 3)

C. (2x + 4) / (x - 3)

D. (3x - 2) / (2x + 2)

E. (3x - 3) / (-2x + 2)

Pembahasan

Terlebih dahulu tentukan f o g(x) dengan cara mengganti x pada f(x) dengan g(x).

Pembahasan Soal invers Fungsi Komposisi

Catatan:

Cara menginvers fungsi pembagian f(x) = (ax + b) / (cx + d) maka f-1(x) = (-dx + b) / (cx - a)

Jawaban: B

. Diketahui (f o g)(x) = x + 1 dan f(x-2) = (x – 1)/(x – 2). Maka nilai dari g-‘ (2) [dibaca: g invers 2] adalah…

Penyelesaian:

f(x – 2) = (x – 1)/(x – 2)

Inverskan x-2 agar ditemukan nilai dari f(x)

y = x – 2

x = y + 2 <—-> y = x + 2 maka:

f(x) = [(x + 2) – 1]/[(x + 2) – 2]

f(x) = (x + 1)/x

(f o g)(x) = f(g(x))

f(g(x)) = x + 1

[g(x) + 1]/g(x) = x + 1

g(x) + 1 = (x + 1). g(x)

g(x) + 1 = x.g(x) + g(x)

g(x) – x.g(x)

-g(x) = -1

-x.g(x) = -1

g(x) = 1/x

g(x) = 1/x

y = 1/x

x = 1/y, maka:

g-‘(x) = 1/x

Jadi, nilai dari g-‘(2) adalah = 1/x = 1/2.

Diketahui fungsi f:R→R dengan f(x)= x²-4x+2 dan fungsi g:R→R dengan g(x)=3x-7. Tentukan:a. (fog) (x)

b. (gof) (x)

Jawaban

5,0/5

Diketahui fungsi f:R → R dengan f(x) = x² - 4x + 2 dan fungsi g:R → R dengan g(x) = 3x - 7.

a. (f o g)(x) = 9x² - 54x + 79

b. (g o f)(x) = 3x² - 12x - 1

c. (f o g)(5) = 34

d. (g o f)(10) = 179

Pembahasan

Diketahui:

f(x) = x² - 4x + 2

g(x) = 3x - 7

Ditanyakan:

a. (f o g)(x) = ...

b. (g o f)(x) = ...

c. (f o g)(5) = ...

d. (g o f)(10) = ...

Penyelesaian:

a. Fungsi (f o g)(x)

Fungsi (f o g)(x) berarti fungsi g(x) dimasukkan ke dalam fungsi f(x).

(f o g)(x) = f(g(x))

(f o g)(x) = (3x - 7)² - 4(3x - 7) + 2

(f o g)(x) = 9x² - 21x - 21x + 49 - 12x + 28 + 2

(f o g)(x) = 9x² - 21x - 21x - 12x + 49 + 28 + 2

(f o g)(x) = 9x² - 54x + 79

Jadi rumus fungsi (f o g)(x) adalah 9x² - 54x + 79.

b. Fungsi (g o f)(x)

Fungsi (g o f)(x) berarti fungsi f(x) dimasukkan ke dalam fungsi g(x).

(g o f)(x) = g(f(x))

(g o f)(x) = 3(x² - 4x + 2) - 7

(g o f)(x) = 3x² - 12x + 6 - 7

(g o f)(x) = 3x² - 12x - 1

Jadi rumus fungsi (g o f)(x) adalah 3x² - 12x - 1.

c. Nilai fungsi (f o g)(5)

(f o g)(x) = 9x² - 54x + 79

(f o g)(5) = 9(5)² - 54(5) + 79

(f o g)(5) = 225 - 270 + 79

(f o g)(5) = 34

Jadi nilai fungsi (f o g)(5) adalah 34.

d. Nilai fungsi (g o f)(10)

(g o f)(x) = 3x² - 12x - 1

(g o f)(10) = 3(10)² - 12(10) - 1

(g o f)(10) = 300 - 120 - 1

(g o f)(10) = 179

Jadi nilai fungsi (g o f)(10) adalah 179.

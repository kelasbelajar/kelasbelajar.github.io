2. uraikan kelemahan dari format bmp​


1. Ukuran file yang besar: BMP adalah format file gambar lossless dan menyimpan informasi yang lengkap. Hal ini berarti bahwa ukuran file BMP dapat menjadi sangat besar, terutama untuk gambar yang lebih besar.

2. Tidak kompatibel dengan beberapa perangkat: BMP tidak didukung oleh beberapa perangkat seperti iPhone dan iPad.

3. Tidak dapat dikompresi: Karena BMP adalah format lossless, gambar tidak dapat dikompresi untuk mengurangi ukurannya.

4. Rendah dalam kualitas gambar: Karena BMP tidak menyimpan informasi yang dikompresi, gambar dapat terlihat rendah dalam kualitas.

6. Jelaskan perintah Tugas yang harus dikerjakan oleh Mahasiswa serta ketentuan pengiriman tugas seperti tugas harus diunggah menggunakan format doc atau docx dsb.


Pembahasan

Perintah tugas yang harus dikerjakan oleh mahasiswa sangatlah beragam bentuknya. Ada beberapa bentuk tugas yang dilakukan sama persis seperti yang dilakukan oleh para siswa dibawah tingkatan mahasiswa seperti Tugas Rumah (PR), tugas kelas seperti mengisi soal-soal latihan dan sebagainya. Perbedaan yang paling menonjol dalam tugas yang dikerjakan oleh mahasiswa dibandingkan dengan siswa-siswi dibawah tingkatannya adalah "Presentasi". Walaupun mungkin dalam tingkatan SMP/SMA ada tugas yang mengharuskan para murid melakukan presentasi, namun frekuensi keseringannya sangat jauh berbeda dengan ketika seseorang sudah masuk ke jenjang pendidikan perkuliahan.

Presentasi merupakan suatu kegiatan berbicara di hadapan banyak hadirin atau salah satu bentuk komunikasi dimana sang pembicara melakukan kegiatan tersebut secara individu atau dalam kelompok. Presentasi merupakan kegiatan pengajuan suatu topik, pendapat atau informasi kepada orang lain yang dalam hal ini adalah dosen dan para mahasiswa lain.

Selain presentasi, untuk jurusan-jurusan perkuliahan tertentu seperti jurusan teknik contohnya, terdapat tugas khas yang berupa tugas praktek. Untuk jurusan tata boga contohnya, para mahasiswa akan sering diberikan tugas untuk melakukan suatu uji coba pembuatan hidangan tertentu sesuai dengan teori yang sudah mereka terima dari sang dosen.

Untuk ketentuan pengiriman tugas, tidak ada standard khusus yang digunakan secara umum. Setiap tugas memiliki metode pengiriman yang berbeda-beda. Untuk tugas seperti makalah atau soal-soal latihan, tugas tersebut bisa dikirimkan melalui email (dengan format apapun sesuai permintaan sang dosen) atau bisa langsung dikumpulkan ke meja kerja sang dosen dengan format print-out. Untuk tugas-tugas semacam praktek, hasil tugas biasanya akan diamati secara langsung oleh sang pendidik dan penilaian juga akan dilakukan secara langsung pada saat prosesi praktek.

13. Uraikanlah jenis-jenis format buku digital dan aplikasi yang digunakan untuk membukanya​.


Jawaban:

1. MOBI - Format MobiPocket

Merupakan format buku digital yang khusus dikembangkat untuk smartphone yang mendukung Windows dan perangkat PDA

2. PDB - Palm File Database

Untuk buku digital berformat PalmDoc (AportisDoc) dan format e-Reader

3. CHM - Compressed HTML

Sering digunakan untuk file bantuan Windows

4. AZW - Amazon Word

Digunakan untuk Kindle Amazon

5. KF8 - Format Kindle Fire

Dari Amazon disusun dalam pembungkus Palm File Database (PDB) dengan Digital Right Management (DMR) Amazon

6. PDF - Portable Document Format

Diciptakan oleh Adobe untuk produk Akrobat mereka dan mendukung trasnfer dokumen

7. HTML - Hyper Text Markup Language

Adalah tulang punggung dari World Wide Web (www)

8. XHTML

Merupakan format yang standar untuk data epub. XHTML merupakan versi khusus dari HTML yang dirancang agar sesuai dengan aturan konstruksi XML.

9. XML

Memiliki tujuan umum markup language untuk pertukaran data

10. EPUB - Elektronik Publication

Mengacu pada standar XHTML dan XML

11. PRC - Palm Resource File

Sering menyertakan alat baca Mobipocket tetapi kadang menyertakan e-Reader atau alat baca ApostisDoc


14. Uraikan yang Anda ketahui tentang definisi format atau ekstensi file menurut terminologinya!​


Jawaban:

Untuk membedakan file yang satu dengan yang lainnya

Penjelasan:

Introduction :

Sebenarnya definisi format atau ekstensi file itu akan berbeda beda karena setiap format memiliki kekurangan dan kelebihan masing masing. Setiap format juga dirancang dengan environment yang berbeda-beda yang akan kita bahas di selanjutnya

Descriptive :

Setiap format dirancang dengan struktur yang berbeda-beda dan bisa dikatakan sesuai dengan penggunaannya.

   1. Extension : Purpose Based :

 Ekstensi file seperti sugiono.3gp atau .mp4 terbuat dengan base-code yang berbeda-beda. Sebagai contoh, format .mp4 merupakan hasil dari H264 yang telah di-"perbarui" sekaligus mencampurkan FFMPEG di dalamnya. Format video yang lain yaitu .mkv yang mendukung format video LossLess seperti x264. Begitu pula untuk kebanyak format file lainnya

   2. Extension Boundary :

 Ekstensi tipe ini merupakan dan hanya diperuntukan untuk file penyimpanan dari software tertentu saja. Biasanya hal ini digunakan untuk melindungi konten file yang dihasilkan software tersebut tidak terbaca oleh software yang lainnya atau juga untuk meng-enkripsi konten tersebut sehingga hanya bisa terbaca oleh file tertentu saja. Yang tentunya anda harus mendownload software yang diperlukan untuk membuka format tersebut. Ekstensi tipe ini biasanya digunakan untuk memonopoli karena memaksa anda untuk mendownload software tersebut untuk membukanya.


15. uraikan yang kalian ketahui tentang ekstensi format gambar​


Ekstensi format gambar adalah akhiran atau tambahan ke sebuah nama file yang menentukan jenis format gambar yang digunakan dalam file tersebut. Ekstensi adalah cara untuk membedakan berbeda jenis gambar ini dan membantu komputer untuk mengetahui bagaimana untuk membaca file gambar yang dimaksud.

Berikut adalah beberapa contoh ekstensi format gambar paling populer:

jpg (JPEG)png (Portable Network Graphics)gif (Graphics Interchange Format)tif (Tagged Image Format)bmp (Bitmap)

Ekstensi format gambar membantu komputer dan perangkat lunak lain untuk mengetahui bagaimana untuk membaca file gambar dengan benar. Pada kenyataannya, file gambar yang sebenarnya adalah sekumpulan data yang disimpan dalam format tertentu. Ekstensi format gambar membantu komputer dalam menyingkirkan semua data yang tidak diperlukan dan memastikan gambar dapat dibaca dengan benar

37. apa tugas dari format dalam permainan basket?


Posisi Pemain Bola Basket antara lain :1.      Point Guard (PG): Tugasnya adalah membawa bola dan memberikan passing pada temannya. Dia yang mengatur irama permainan timnya, apakah cepat atau lambat. Biasa juga disebut playmaker. Posisi ini biasanya dipegang oleh orang yang berbadan kecil tapi lincah dan jago mendribel.
2.      Shooting Guard (SG): Seperti namanya, SG adalah orang yang punya tugas untuk menembak bola dari jarak-jarak yang cukup jauh. Dia harus cepat bergerak dan mencari posisi yang kosong untuk melepaskan tembakan. Posisi ini biasanya dipegang oleh orang yang paling bagus akurasi shootingnya.
3.      Small Forward (SF): SF punya tugas mencetak angka. Dengan kata lain seorang SF harus mampu menerobos pertahanan dan melakukan lay-up atau dunk, juga melakukan shoot dari jarak-jarak tertentu. Posisi ini biasanya dipegang oleh orang yang punya tekhnik hebat dan jago mencetak angka.
4.      Power Forward (PF): PF tugasnya adalah merebound bola. Rebound sangat penting, jika dalam pertahanan, rebound penting agar bola tak kembali ke tangan musuh. Jika dalam penyerangan, rebound penting untuk kembali menciptakan kesempatan membuat angka. Biasanya posisi ini dipegang oleh orang yang lompatannya tinggi dan badannya cukup besar untuk beradu fisik.
5.      Center (C): C tugasnya di pertahanan dan penyerangan.. Pada pertahanan, ia harus mampu mengamankan ringnya dari tembakan jarak dekat seperti lay-up atau dunk musuh.. Pada penyerangan, C harus mampu melihat posisi teman-temannya dan memberikan umpan pada teman yang kosong (karena posisinya di tengah, sehingga mudah memberi umpan ke sisi manapun).. Selain itu dia pun harus kuat beradu fisik dengan musuh untuk mencetak angka di bawah ring.. Posisi ini biasanya dipegang oleh orang yang badannya paling tinggi dan besar.

Saya sengaja berangkat lebih pagi karena hari itu sekolah kami akan mengadakan upacara bendera untuk pertama kali. 

Tiba di halaman sekolah, saya melihat anak-anak sudah mulai ramai. Tetapi ketika saya masuk ke ruangan guru, cuma terlihat seorang guru. Saya melirik jam. Sudah hampir pukul delapan. Hampir tiga bulan saya mengajar di sekolah itu, belum pernah sekalipun saya melihat semua guru datang tepat waktu. Hari itu, kami akhirnya kembali gagal mengadakan upacara bendera.

Karena sekolah baru menyelesaikan pembangunan ruang kelas baru, maka jadwal hari itu adalah beres-beres ruangan. Di antara rasa lelah, saya mendengar tiga murid perempuan memanggil dengan suara setengah berbisik dari pintu kelas, “Pak Ridwan…. Pak!”

“Ada apa?” tanyaku pelan.

“Sini, Pak!” panggil mereka lagi. 

Saya pun berjalan menghampiri mereka, dan bertanya “Ada apa?”
    
Salah seorang di antara mereka lalu menyodorkan secarik kertas. “Surat untuk Bapak. Dibaca, ya, Pak!” pinta mereka. Ketiganya kemudian bergegas lari sambil tertawa kecil. 
Saya membuka lembaran tersebut dan membaca: Pak Ridwan ganteng sama Pak Gilang lucu sekali dan imut.
    
Gilang — sahabat saya dari Indonesia Mengajar — tersenyum ketika saya menunjukan surat itu. Beberapa saat kemudian, ketiga anak tadi muncul dan sekali lagi menyodorkan secarik kertas kepada saya. “Yang ini harus dibalas, ya, Pak!” kata mereka sambil berlari. 

Begini isi surat itu: Pak Gilang dan Pak Ridwan nanti malam sholat, ya, Pak. Ngaji, Pak. Assalamuallaikum, baca, ya, Pak. Dibalas. Awas kalau tidak dibalas, nanti ada balasannya di masjid.
    
Menemani anak-anak mengaji merupakan tugas tidak wajib bagi kami para pengajar muda. Namun justru di waktu itulah kami bisa menjalin keakraban dengan mereka. Maklum, kadang anak-anak sangat tertutup saat berada di sekolah sehingga kami juga sering kesulitan mengetahui masalah yang sebenarnya, ketika ada salah satu anak didik yang tiba-tiba berubah sifat dan menjadi malas belajar. Padahal, tujuan dari Indonesia Mengajar adalah untuk memperbanyak porsi anak agar lebih aktif di kelas. 
    
Untuk menyiasati itu, saya bahkan menyelenggarakan program Buku Curhat (curahan hati). Setiap anak diperbolehkan menuliskan segala unek-unek dan masalah mereka di buku tersebut dan dikumpulkan ke saya. Hasilnya mengejutkan! Banyak permasalahan mereka — entah itu masalah keluarga atau dalam berteman — yang saya tidak ketahui.

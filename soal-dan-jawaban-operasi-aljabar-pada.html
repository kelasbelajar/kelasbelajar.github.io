a = -3.b =9.c = 2.d = -2.

Operasi matriks pada soal merupakan operasi penjumlahan matriks di mana setiap matriks memiliki dua kolom dan dua baris.

Penjelasan dengan langkah-langkah

Matriks ruas 1, ruas 2, dan ruas 3 pada soal adalah matriks 2 x 2. Maka ketiga matriks memiliki 2 kolom dan 2 baris. Operasi matriks pada soal adalah penjumlahan dan kemudian penyetaraan nilai samping kiri dan kanan tanda sama dengan.

Diketahui:

[tex]\left[\begin{array}{ccc}\sf 2c-3&\sf 8\\\sf -1&\sf 4d+1\end{array}\right] + \left[\begin{array}{ccc}\sf d&\sf 1\\\sf -2&\sf c+5\end{array}\right] =\left[\begin{array}{ccc}\sf -1&\sf b\\\sf a&\sf 0\end{array}\right][/tex]

Penjumlahan matriks di atas dimisalkan sebagai:

Matriks A + Matriks B = Matriks C.

Ditanyakan:

Penyelesaian aljabar ⇒

a = ?b = ?c = ?d = ?

Penyelesaian:

Langkah 1
Penyetaraan nilai di kolom kedua baris pertama.

A(1, 2) + B (1, 2) = C (1, 2)
8 + 1 = b
b = 9

Langkah 2
Penyetaraan nilai di kolom pertama baris kedua.

A(2, 1) + B (2, 1) = C (2, 1)
-1 + (-2) = a
a = -3

Langkah 3
Penyetaraan nilai di kolom pertama dan baris pertama.

A(1, 1) + B(1, 1) = C(1, 1)
2c - 3 + d = -1
2c + d = -1 + 3
2c + d = 2
d         = 2 - 2c ...(1)

Langkah 4
Penyetaraan nilai di kolom kedua dan baris kedua.

A(2, 2) + B(2, 2) = C(2, 2)
4d + 1 + c + 5 = 0
4d + c = -6 ...(2)

Langkah 5
Perhitungan nilai c dan d.

Substitusi persamaan (1) ke (2):
4 (2 - 2c) + c = -6
8 - 8c + c = -6
-7c = -14
c = 2Substitusi nilai c ke persamaan (1):
d = 2 - 2(2)
d = 2 - 4
d = -2

Berikut adalah beberapa rumus yang digunakan dalam materi vektor.

Perkalian skalar (dot product) antara dua vektor, jika diketahui besar sudut antara dua vektor tersebut adalah sebagai berikut.

a · b = |a| |b| cos θ

Besar cosinus sudut antara dua vektor dirumuskan sebagai berikut.

cos θ = (a · b)/(|a| |b|)

|a + b| = √(|a|² + |b|² + 2 |a| |b| cos θ)|a - b| = √(|a|² + |b|² - 2 |a| |b| cos θ)

Keterangan:

a = vektor a

b = vektor b

|a| = panjang vektor a

|b| = panjang vektor b

θ = besar sudut antara vektor a dan vektor b

↓↓↓

Diketahui:

|a| = 10

|b| = 6

∠(a, b) = θ = 60°

Ditanyakan:

|2a - b| = ...

Penyelesaian:

Vektor 2a merupakan perpanjangan dari vektor a. Karena itu, sudut yang terbentuk oleh vektor 2a dan b sama dengan sudut yang terbentuk oleh vektor a dan b. Dengan menggunakan aturan cosinus vektor, diperoleh persamaan sebagai berikut.

|2a - b| = √(|2a|² + |b|² - 2 |2a| |b| cos θ)

|2a - b| = √((2(10))² + 6² - 2 (2(10))(6) cos 60°)

|2a - b| = √(20² + 6² - 2 (20)(6)(½))

|2a - b| = √(400 + 36 - 120)

|2a - b| = √316

|2a - b| = √(4 × 79)

|2a - b| = 2√79

Jadi, jika |a| = 10, |b| = 6, dan ∠(a, b) = 60°, maka |2a - b| = 2√79.

17. bantu aku ya kak Besar sudut BCA​


Jawab:

D. 60°

Penjelasan dengan langkah-langkah:

∠BCA:

cari x terlebih dahulu

x=>

180° = 3x - 30°

180° - 30° = 3x

150° = 3x

[tex]\frac{150}{3}[/tex] = x

x = 50°.

Kemudian cari sudut BCA

∠BCA = 180° - (3x - 30°)

          = 180° - ((3.50°)-30°)

          = 180° - (150°-30°)

          = 180° - 120°

          = 60°

20. Besar<BCA adalahtolong bantu plis​


[Sudut ABC = 180° - 120° = 60°]

Sudut ABC + Sudut BCA + Sudut CAB = 180° <=> 60° + (3x + 10)° + (x + 10)° = 180° <=> 60° + 10° + 10° + 3x + x = 180° <=> 80° + 4x = 180° <=> 4x = 180° - 80° = 100° <=> x = 100/4 = 25°.

Sudut BCA = (3x + 10)° = 3(25) + 10 = 75 + 10 = 85°.

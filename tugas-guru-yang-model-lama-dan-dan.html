Kerucut
d = 42 cm -> r = 21 cm
t = 28 cm
L karton yang dibutuhkan?

r^2 + t^2 = s^2
21^2 + 28^2 = s^2
441 + 784 = s^2
1.225 = s^2
35 cm = s

Jika kerucut dengan alas, maka :
L kerucut = L alas + L selimut
= (phi x r^2) + (phi x r x s)
= phi x r x (r + s)
= 22/7 x 21 x (21 + 35)
= 66 x 56
= 3.696 cm^2

Jika kerucut tanpa alas, maka :
L kerucut = L selimut
= phi x r x s
= 22/7 x 21 x 35
= 66 x 35
= 2.310 cm^2

L = [tex]\frac{1}{2}[/tex] × 14 × 18 = 126 cm²

maka volume limas adalah:

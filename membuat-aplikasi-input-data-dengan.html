import java.util.Scanner;

public class InputDataSekolah {

   public static void main(String[] args) {

       Scanner input = new Scanner(System.in);

       

       System.out.print("Masukkan nama sekolah: ");

       String namaSekolah = input.nextLine();

       

       System.out.print("Masukkan alamat sekolah: ");

       String alamatSekolah = input.nextLine();

       

       System.out.print("Masukkan nomor telepon sekolah: ");

       String teleponSekolah = input.nextLine();

       

       System.out.print("Masukkan jumlah siswa di sekolah: ");

       int jumlahSiswa = input.nextInt();

       

       System.out.println("\nData sekolah:");

       System.out.println("Nama sekolah: " + namaSekolah);

       System.out.println("Alamat sekolah: " + alamatSekolah);

       System.out.println("Nomor telepon sekolah: " + teleponSekolah);

       System.out.println("Jumlah siswa di sekolah: " + jumlahSiswa);

   }

}

Program ini akan meminta pengguna untuk memasukkan data sekolah seperti nama, alamat, nomor telepon, dan jumlah siswa di sekolah. Kemudian program akan menampilkan kembali data tersebut sebagai output.

Penjelasan:

Berikut adalah langkah-langkah untuk membuat data, dari membuat data manual hingga menginput data dan menyimpan data di dalam komputer:

Membuat Data Manual

Jika data yang ingin dibuat hanya sedikit atau tidak terlalu rumit, maka membuat data secara manual bisa menjadi pilihan. Langkah-langkahnya adalah:a. Tentukan jenis data yang ingin dibuat. Misalnya data nama dan alamat kontak.b. Siapkan kertas atau buku catatan untuk mencatat data.c. Tulis header atau judul data di atas kertas. Misalnya "Daftar Nama dan Alamat Kontak".d. Tambahkan kolom atau field yang sesuai dengan jenis data yang ingin dibuat. Misalnya kolom Nama, Alamat, Nomor Telepon, dan Email.e. Isi data pada setiap kolom sesuai dengan informasi yang dimiliki. Pastikan data yang diisi akurat dan relevan.f. Setelah selesai membuat data, simpan kertas atau buku catatan di tempat yang aman dan mudah diakses.
Menginput Data
Jika data yang ingin dibuat banyak atau kompleks, maka menginput data ke dalam komputer bisa menjadi pilihan. Langkah-langkahnya adalah:a. Tentukan jenis data yang ingin dibuat dan aplikasi apa yang akan digunakan untuk menginput data. Misalnya data penjualan dan menggunakan aplikasi Microsoft Excel.b. Buka aplikasi yang akan digunakan dan buat file baru.c. Buat header atau judul data di atas file. Misalnya "Data Penjualan".d. Tambahkan kolom atau field yang sesuai dengan jenis data yang ingin dibuat. Misalnya kolom Tanggal, Nama Barang, Jumlah, Harga, dan Total.e. Isi data pada setiap kolom sesuai dengan informasi yang dimiliki. Pastikan data yang diisi akurat dan relevan.f. Setelah selesai menginput data, simpan file di dalam komputer dengan nama yang mudah diingat dan tempat yang mudah diakses.
Menyimpan Data di dalam Komputer

Setelah membuat dan menginput data, langkah selanjutnya adalah menyimpan data di dalam komputer. Langkah-langkahnya adalah:

a. Pastikan data yang ingin disimpan telah diinput dengan lengkap dan akurat.

b. Klik menu "File" pada aplikasi yang digunakan.

c. Pilih opsi "Save As".

d. Tentukan nama file dan lokasi penyimpanan file di dalam komputer.

e. Pilih format file yang sesuai dengan aplikasi yang digunakan.

f. Klik tombol "Save" untuk menyimpan file di dalam komputer.

g. Pastikan file telah tersimpan dengan baik dan dapat diakses kembali jika dibutuhkan.

```python

# fungsi untuk mengubah suhu dari Celcius ke Fahrenheit

def celcius_to_fahrenheit(celcius):

return (celcius * 9/5) + 32

# fungsi untuk mengubah suhu dari Fahrenheit ke Celcius

def fahrenheit_to_celcius(fahrenheit):

return (fahrenheit - 32) * 5/9

# menu program

print("=== Konversi Suhu ===")

print("1. Celcius ke Fahrenheit")

print("2. Fahrenheit ke Celcius")

# input pilihan menu

menu = input("Masukkan pilihan menu (1/2): ")

# inisialisasi array data suhu

data_suhu = []

# jika pilihan menu adalah 1

if menu == "1":

# input banyaknya suhu yang akan diubah

n = int(input("Masukkan banyaknya suhu yang akan diubah: "))

# perulangan sebanyak n kali untuk meminta input suhu dalam Celcius

for i in range(n):

suhu_celcius = float(input("Masukkan suhu dalam Celcius: "))

suhu_fahrenheit = celcius_to_fahrenheit(suhu_celcius)

data_suhu.append((suhu_celcius, suhu_fahrenheit))

# jika pilihan menu adalah 2

elif menu == "2":

# input banyaknya suhu yang akan diubah

n = int(input("Masukkan banyaknya suhu yang akan diubah: "))

# perulangan sebanyak n kali untuk meminta input suhu dalam Fahrenheit

for i in range(n):

suhu_fahrenheit = float(input("Masukkan suhu dalam Fahrenheit: "))

suhu_celcius = fahrenheit_to_celcius(suhu_fahrenheit)

data_suhu.append((suhu_fahrenheit, suhu_celcius))

# jika pilihan menu tidak valid

else:

print("Pilihan menu tidak valid")

# menampilkan hasil konversi suhu

for suhu in data_suhu:

print(suhu[0], "derajat", "Celcius" if menu == "1" else "Fahrenheit", "=", suhu[1], "derajat", "Fahrenheit" if menu == "1" else "Celcius")

```

Program di atas akan menampilkan menu untuk memilih konversi suhu dari Celcius ke Fahrenheit atau sebaliknya. Kemudian, program akan meminta input banyaknya suhu yang akan dikonversi, lalu meminta input suhu sebanyak n kali dengan menggunakan perulangan. Setiap suhu yang diinput akan dikonversi menggunakan fungsi yang telah dibuat, dan hasilnya disimpan dalam array `data_suhu`. Terakhir, program akan menampilkan hasil konversi suhu menggunakan perulangan dan array yang telah disimpan sebelumnya.

33. buatkan program c++ untuk membuat fungsi antrian dengan data/nilai yang diinputkan oleh user, setelah selesai di inputkan,maka langsung dihapus satu persatu datanya. PS: Harap gunakan tipe variable ARRAY/STRUCT


Berikut saya cantumkan codenya dalam C++

#include<stdio.h>

#include<stdlib.h>

int main()

{

int jumlahData,*dataNilai;

//Proses Input Jumlah data dan inisialisasi array dinamis

scanf("%d",&jumlahData);

dataNilai = (int*)malloc(sizeof(int)*jumlahData);

//Proses Input Nilai

for(int i=0;i<jumlahData; i++)

{

 scanf("%d",&*(dataNilai+i));

}

//Tampilkan Data Nilai secara Queue

for(int i=0;i<jumlahData; i++)

{

 printf("%d\n",*(dataNilai+i));

}

//Penghapusan data satu per satu dapat menggunakan loop tetapi karena dinamis bisa langsung free  untuk menghindari memory leak

free(dataNilai);

return 0;

}

Jawaban:

Untuk menghitung harga pasar obligasi, kita dapat menggunakan rumus present value (nilai sekarang) dari aliran kas (cash flow) yang akan diterima dari obligasi selama jangka waktu yang tersisa. Berikut adalah langkah-langkahnya:

1. Hitung aliran kas tahunan yang akan diterima dari obligasi:

Aliran kas tahunan =

Nilai nominal obligasi x Bunga obligasi = $1,000 x 10% = $100

2. Hitung present value (nilai sekarang) dari aliran kas tahunan untuk setiap tahun ke depan:

Present value tahun 1 = $100 / (1 + suku bunga pasar) = $100 / (1 + 10%) = $90.91

Present value tahun 2 = $100 / (1 + suku bunga pasar)^2 = $100 / (1 + 10%)^2 = $82.64

Present value tahun 3 = $100 / (1 + suku bunga pasar)^3 = $100 / (1 + 10%)^3 = $75.13

3. Jumlahkan present value dari aliran kas tahunan untuk mendapatkan harga pasar obligasi:


Harga pasar obligasi =

Present value tahun 1 + Present value tahun 2 + Present value tahun 3

= $90.91 + $82.64 + $75.13

= $248.68

Jadi, dengan asumsi suku bunga pasar umum adalah 10% dan jatuh tempo obligasi 3 tahun mendatang, harga pasar obligasi tersebut sebesar $248.68.

27. Bila anda berinvestasi pada surat berharga berpenghasilan tetap yaitu obligasi dengan nominal sebesar $1.000.,bunga obligasi yang berlaku 10% per tahun, pembayaran bunga obligasi setiap tahunan. Bila suku bunga pasar umum adalah 10% berapakah harga pasar obligasi, jika jatuh tempo obligasi 3 tahun mendatang.mohon bantuannya kaks ​


Jawaban:

Untuk menghitung harga pasar obligasi, kita dapat menggunakan rumus present value (nilai sekarang) dari aliran kas (cash flow) yang akan diterima dari obligasi selama jangka waktu yang tersisa. Berikut adalah langkah-langkahnya:


1. Hitung aliran kas tahunan yang akan diterima dari obligasi:

Aliran kas tahunan =
Nilai nominal obligasi x Bunga obligasi = $1,000 x 10% = $100

2. Hitung present value (nilai sekarang) dari aliran kas tahunan untuk setiap tahun ke depan:

Present value tahun 1 = $100 / (1 + suku bunga pasar) = $100 / (1 + 10%) = $90.91

Present value tahun 2 = $100 / (1 + suku bunga pasar)^2 = $100 / (1 + 10%)^2 = $82.64

Present value tahun 3 = $100 / (1 + suku bunga pasar)^3 = $100 / (1 + 10%)^3 = $75.13

3. Jumlahkan present value dari aliran kas tahunan untuk mendapatkan harga pasar obligasi:


Harga pasar obligasi =

Present value tahun 1 + Present value tahun 2 + Present value tahun 3

= $90.91 + $82.64 + $75.13

= $248.68

Jadi, dengan asumsi suku bunga pasar umum adalah 10% dan jatuh tempo obligasi 3 tahun mendatang, harga pasar obligasi tersebut sebesar $248.68.

12. Sebuah perusahaan yang memproduksi makanan kaleng memberikan tunjangan kepada karyawannya selain gaji bulanan. di perusahaan tersebut, setiap bulannya seorang karyawan akan mendapatkan dua macam tunjangan yaitu tunjangan keluarga dan tunjangan kesehatan. besarnya tunjangan keluarga ditentukan dari 10% dari gaji pokok ditambah rp100.000,00 setiap 1 anak. sementara besarnya tunjangan kesehatan adalah setengah dari tunjangan keluarga. seorang karyawan memiliki 2 anak, jika gaji pokoknya adalah rp4.000.000,00 maka berdasarkan situasi tersebut pendapatan tiap bulan karyawan tersebut adalah?


Pendapatan karyawan tersebut setiap bulan adalah Rp4.900.000,00.

Untuk mengerjakan soal ini kita harus mengetauhi rumus dari ekonomi yaitu besarnya pendapatan dan pengeluaran.

Diketauhi :

Tunjangan keluarga karena memiliki 2 anak: Rp100.000,00 x 2 = Rp200.000,00Total tunjangan keluarga: Rp400.000,00 + Rp200.000,00 = Rp600.000,00

Ditanya : pendapatan tiap bulan karyawan tersebut adalah?

Penjelasan dengan langkah-langkah :

Untuk mengerjakan soal ini kita harus mengetauhi rumus dari ekonomi yaitu besarnya pendapatan dan pengeluaran.

Besarnya tunjangan keluarga dapat dihitung sebagai berikut:

10% x Rp4.000.000,00 = Rp400.000,00

Tunjangan keluarga karena memiliki 2 anak: Rp100.000,00 x 2 = Rp200.000,00Total tunjangan keluarga: Rp400.000,00 + Rp200.000,00 = Rp600.000,00

Besarnya tunjangan kesehatan adalah setengah dari tunjangan keluarga, sehingga:

Tunjangan kesehatan = 1/2 x Rp600.000,00 = Rp300.000,00

Total pendapatan karyawan setiap bulan adalah gaji pokok ditambah tunjangan keluarga dan tunjangan kesehatan:

Rp4.000.000,00 + Rp600.000,00 + Rp300.000,00 = Rp4.900.000,00

Jadi, pendapatan karyawan tersebut setiap bulan adalah Rp4.900.000,00.

19. keluarga dan tunjangan kesehatan. Besarnya tunjangan keluarga ditentukan dari 10% dari gaji pokok ditambah Rp100.000,00 setiap 1 anak. Sementara besarnya tunjangan kesehatan adalah setengah dari tunjangan keluarga. Seorang karyawan memiliki 2 anak, jika gaji pokoknya adalah maka Rp4.000.000,00 berdasarkan situasi tersebut tiap bulan pendapatan karyawan tersebut adalah ....​


Jawaban:

Pendapatannya sebesar Rp 2.740.000

Penjelasan dengan langkah-langkah:

Tunjangan Keluarga

10% × (4.000.000 + 200.000)

10/100 × 4.200.000

10 × 42.000

420.000 × 2 (karena ada 2 anak) = Rp 840.000

Tunjangan Kesehatan

½ × Rp 840.000 = Rp 420.000

Pendapatan Karyawan

Rp 4.000.000 - (Rp 840.000 + Rp 420.000)

Rp 4.000.000 - Rp 1.260.000

Diketahui:* • Pegawai 1 Ahmad muslim memiliki gaji pokok Rp.4.000.00 dan memiliki dua orang anak. • Pegawai 2 Alex beragama Kristen Protestan memiliki gaji pokok Rp. 6.000.000 dan memiliki 5 orang anak. *Kalkulasi:* 1. Gaji Bersih = Gaji Pokok + Tunjangan Jabatan + Tunjangan Keluarga *Soal:* • Buat aplikasi sederhana dengan pemrograman python untuk mencetak : • Nama pegawai, agama, gaji pokok, tunjangan jabatan, tunjangan keluarga, gaji bersih (take home pay).TOLONG BANTU PYHTON ​


Jawaban:

Berikut adalah contoh program sederhana dalam bahasa pemrograman Python yang menghitung tunjangan jabatan, tunjangan keluarga, dan gaji bersih berdasarkan informasi yang diberikan:

```python

# Informasi pegawai

pegawai1 = {

'nama': 'Ahmad Muslim',

'agama': 'Islam',

'gaji_pokok': 4000000,

'jumlah_anak': 2

}

pegawai2 = {

'nama': 'Alex',

'agama': 'Kristen Protestan',

'gaji_pokok': 6000000,

'jumlah_anak': 5

}

# Fungsi untuk menghitung tunjangan jabatan

def hitung_tunjangan_jabatan(gaji_pokok):

return 0.2 * gaji_pokok

# Fungsi untuk menghitung tunjangan keluarga

def hitung_tunjangan_keluarga(gaji_pokok, jumlah_anak):

if jumlah_anak <= 2:

return 0.1 * gaji_pokok

else:

return 0.2 * gaji_pokok

# Fungsi untuk menghitung gaji bersih

def hitung_gaji_bersih(gaji_pokok, tunjangan_jabatan, tunjangan_keluarga):

return gaji_pokok + tunjangan_jabatan + tunjangan_keluarga

# Menampilkan informasi pegawai dan hasil perhitungan

def cetak_info_pegawai(pegawai):

print(f'Nama Pegawai: {pegawai["nama"]}')

print(f'Agama: {pegawai["agama"]}')

print(f'Gaji Pokok: Rp. {pegawai["gaji_pokok"]:,}')

tunjangan_jabatan = hitung_tunjangan_jabatan(pegawai["gaji_pokok"])

print(f'Tunjangan Jabatan: Rp. {tunjangan_jabatan:,}')

tunjangan_keluarga = hitung_tunjangan_keluarga(pegawai["gaji_pokok"], pegawai["jumlah_anak"])

print(f'Tunjangan Keluarga: Rp. {tunjangan_keluarga:,}')

gaji_bersih = hitung_gaji_bersih(pegawai["gaji_pokok"], tunjangan_jabatan, tunjangan_keluarga)

print(f'Gaji Bersih (Take Home Pay): Rp. {gaji_bersih:,}')

# Menampilkan informasi pegawai

print("Informasi Pegawai 1:")

cetak_info_pegawai(pegawai1)

print("\nInformasi Pegawai 2:")

cetak_info_pegawai(pegawai2)

```

Program ini akan menghitung tunjangan jabatan, tunjangan keluarga, dan gaji bersih untuk kedua pegawai berdasarkan informasi yang diberikan dan kemudian mencetak hasilnya. Pastikan Anda menjalankan program ini di lingkungan Python yang sudah terinstal di komputer Anda.

24. buatlah persamaan kuadrat dari x²-x-20=0 melalui 3 cara​


Persamaan Kuadrat

====================

Jawaban:

5 atau -4

====================

Penjelasan dan langkah-langkah:

Cara Faktorisasi

x² - x - 20      = 0

(x -5) × (x + 4) = 0

                   x = 5 atau -4

Cara Melengkapkan Kuadrat

x² - x - 20      = 0

x² - x + (-1/2)² = 20 + (-1/2)²

(x - 1/2)²         = 81/4

x - 1/2             = ±√81/4

x - 1/2             = ±9/2

x                     = 5 atau -4

Cara Rumus Kuadrat

x² - x - 20 = 0

x               = (-b ± √b² - 4ac) / 2a

x               = (1 ± √1 - 80) / 2

x               = (1 ± 9)/2

x               = 5 atau -4

====================

29. Buatlah grafik fungsi kuadrat berikut ini ! F(x)=x²+x-12 (buat caranya)


Grafik fungsi kuadrat berikut ini bisa di lihat pada gambar di lampiran

F(x)=x²+x-12

PEMBAHASAN

Sebelumnya marilah kita pelajari dulu prinsip-prinsip dari fungsi kuadrat sebagai berikut ini.

Fungsi kuadrat memiliki bentuk umum y = ax² + bx + c

dimana :

x = -b/2a → persamaan sumbu simetri

y = ( b² - 4ac ) / (-4a) → nilai maksimum fungsi

Okay dengan prinsip ini kita akan mencoba selesaikan soalnya.

Diketahui fungsi kuadrat f(x)=x²+x-12

Untuk menggambarkan fungsi kuadrat ini dapat dicari beberapa titik penting sebagai berikut :

1. Titik potong dengan sumbu x

f(x)=x²+x-12=0

(x+4)(x-3)=0

x = -4 atau x = 3

Titik potong dengan sumbu x adalah di ( -4,0 ) dan ( 3,0 )

2. Titik potong dengan sumbu y

f(x) = x²+x-12

f(0) =  0²+0-12

f(0) = -12

Titik potong dengan sumbu y adalah di (0,-12)

3. Titik balik

x = -b/2a = -1 / 2(1) = -1/2 = -0,5

y = ( b² - 4ac ) / (-4a) = ( 1² - 4(1)(-12) ) / (-4(1)) = -12,25

Titik Balik Minimum adalah di ( -0,5 , -12,25 )

Selanjutnya kita gambarkan titik-titik penting ini di koordinat kartesius dan akan di dapatkan gambar seperti pada lampiran.

Jawaban:

Untuk mencari hasil bagi dari ekspresi 4x² + 6x dibagi oleh 2x, kita dapat membagi setiap suku dengan 2x. Berikut adalah cara pengerjaannya:

4x² / 2x = 2x (karena 4x² dibagi 2x sama dengan 2x)

6x / 2x = 3 (karena 6x dibagi 2x sama dengan 3)

Jadi, hasil bagi dari ekspresi 4x² + 6x dibagi oleh 2x adalah 2x + 3.

Comand Promp (CMD) adalah program Windows yang bertindak sebagai interpreter perintah DOS. Banyak sekali perintah-pertintah yang bisa dijalankan di CMD. Kegunaannya antara lain : menjalankan berbagai program lewat perintah, menjalankan file ekstensi java, mengecek IP, dan masih banyak lagi.


Disk Defragmenter

Disk Defragmenter adalah salah satu tool yang disediakan oleh Windows yang digunakan untuk menganalisa volume drive, dan untuk mengalokasikan file dan folder yang terpisah.


Notepad

Notepad adalah suatu software kecil yang terbawa secara otomatis saat windows diinstal kekomputer kita. Software ini memilik banyak sekali kegunaan, mulai dari membuat catatan-catatan kecil di komputer, hingga listing program bisa menggunakan notepad.


Remote Dekstop Connection

Remote Dekstop Connection adalah sebuah software berbasis desktop yang kegunaannya seperti remote. Dengan aplikasi ini, kita bisa mengakses banyak komputer lain dari satu komputer induk dalam satu LAN.

24. apakah floating app terdeteksi di exam browser?​


Jawaban:

Saya mohon maaf, namun saya tidak memiliki informasi terbaru mengenai Exam Browser yang digunakan untuk ujian online. Perlu dicatat bahwa kebijakan dan kemampuan deteksi Exam Browser dapat berbeda-beda tergantung pada versi yang digunakan dan aturan yang ditetapkan oleh penyelenggara ujian.

Dalam beberapa kasus, Exam Browser dapat dirancang untuk membatasi akses ke aplikasi atau fitur lain di luar browser ujian untuk menjaga integritas dan keamanan ujian. Ini termasuk kemampuan untuk mendeteksi aplikasi yang berjalan dalam mode mengambang (floating app) dan mungkin melarang penggunaannya.

Namun, untuk informasi yang lebih akurat dan terbaru, disarankan untuk menghubungi penyelenggara ujian atau pihak yang bertanggung jawab atas Exam Browser yang digunakan. Mereka akan memberikan informasi yang lebih spesifik mengenai kebijakan dan kemampuan deteksi Exam Browser yang digunakan dalam konteks ujian tertentu.

1. Microsoft Internet Explorer Image
Internet Explorer atau yang biasa di singkat IE/MSIE ini terintergrasi dengan Sistem Operasi Windows pada umumnya,(jadi kalo Windows anda tidak kenal IE kebangetan banget).Jadi jangan heran kalau penggunanya banyak banget.Versi yang terakhirnya adalah Versi 8 Beta 1(Masih Uji Coba).Web Browser ini di buat oleh Microsoft kepunyaan paman Bill Gates. Kelemahannya yaitu nggak bisa jalan selain di Windows.
2. Mozilla Firefox Image
Inilah pesaing berat Internet Explorer ,Firefox.Si Rubah Api ini di buat oleh Mozilla Foundation yang bertujuan untuk software terbuka atau gratis bagi siapapun. Tersedia dalam 50 lebih bahasa termasuk indonesia smile emotikon.Di lengkapi oleh addons atau aplikasi tambah untuk membantu kinerja si rubah api ini. Juga ada versi portablenya.
3. Netscape Navigator
Web Browser yang pernah terkenal di era tahun 90-an yang paling banyak digunakan sebelum ada IE. Sekitar tahun 1999 Netscape Corporation di akusisi oleh American Online (AOL).Setelah itu Netscape Corporation mendirikan Mozilla Foundation.Jadi bisa diambil kesimpulan bahwa Netscape Navigator adalah browser web yang merupakan gagasan bagi browser Mozilla Firefox.
4. Opera Image
Opera Mini merupakan web browser untuk handphone yang terkenal hingga sekarang. Opera berfitur banyak sekali. Selain gratis browser ini mempunyai versi portablenya. 
5. Safari Image
Safari di produksi oleh Apple Inc (yang meroduksi iPhone dan iPod itu lho).Awalnya Safari dikeluarkan untuk Sistem Operasi Mac saja namun sekitar bulan Juni 2007 di buat untuk Windows.
 6. Google Chrome Image
Inilah Software terbaru dari Google yang di keluarkan pada tanggal 2 September 2008 untuk Windows Xp. Baru-baru ini plugin chrome dibuat. Browser yang mempunyai tema biru putih ini berukuran 474 kb.

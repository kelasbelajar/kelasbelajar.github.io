12. tolong bantu nanti aku tf pulsa 20k​


Jawaban:

Jawabannya adalah suku pertama = -11/2, beda = 1/2 dan rumus suku ke-n adalah Un = n/2 - 6

Konsep barisan aritmatika :

Un = a + (n-1)b

Keterangan :

Un = suku ke-n

a = U1 = suku pertama

b = beda

Jawab :

suku ke−3=−4 1/2 -> a + 2b = - 9/2

suku ke - 8=−2 -> a + 7b = -2

Eliminasi kedua persamaan

a + 2b = -9/2

a + 7b = -2

---------------- (-)

___-5b = -5/2

b = -5/2 x 1/-5

b = 1/2

Substitusi b = 1/2 ke salah satu persamaan

a + 2(1/2) = -9/2

a + 1 = -9/2

a = -9/2 - 1

a = -9/2 - 2/2

a = -11/2

Rumus suku ke-n

Un = a + (n-1)b

= -11/2 + (n-1)1/2

= -11/2 + n/2 - 1/2

= n/2 - 12/2

= n/2 - 6

Jadi suku pertama = -11/2, beda = 1/2 dan rumus suku ke-n adalah Un = n/2 - 6

25. bantu yahh... kalo ada yang bisa bantu aku kasih reward pulsa


massa NH4Cl : 2,675 gram
Volume larutan : 500 mL
kb NH4Cl : 10^-5
pH larutan : ........ ?

jawab

step 1
Tentukan molaritas NH4Cl

M : mol / L

pertama :
Cari mol NH4Cl

mol : massa / Mr

Mr NH4Cl : 14 + 1(4) + 35,5
Mr NH4Cl : 53,5 g/mol

mol NH4Cl : 2,675 g / 53,5 g/mol
NH4Cl : 0,05 mol

kedua :
konversi mL ke L

500 mL x 10^-3 L/mL : 0,5 L

Ketiga :
Subtitusikan data yg diperoleh ke satuan mol/L

M NH4Cl : mol / L
M NH4Cl : 0,05 mol / 0,5 L
NH4Cl : 0,1 M

step 2
Tentukan sifat keasaman dari NH4Cl apakah asam atau basa.

Reaksi Penguraian :
NH4Cl → NH4+ + Cl-

ion NH4+ direaksikan dengan air sehingga akan diketahui keasamannya.

Reaksi
NH4+ H2O → NH4OH + H+

Dari data tersebut terlihat bahwa ada ion H+ yg dihasilkan sehingga sifat garam tersebut adalah asam.

step 3
Tentukan konsentrasi H+

[H+] √(kw / kb) x [garam]
[H+]: √(10^-14 / 10^-5) x 0,1
[H+] : 10^-5

step 4
Tentukan pH garam tersebut.

pH : -log [H+]
pH : -log 10^-5
pH : 5

Pembahasan

1.  Dapatkan hasil dari operasi perpangkatan berikut ini.

[tex]\displaystyle \frac{64^{2} + 16^{3}}{4^{5}}[/tex] = [tex]\displaystyle \frac{(2^{6})^{2} + (2^{4})^{3}}{(2^{2})^{5}}[/tex]

88 + 58 = 146 ✓

Langkah-langkah

Kita gunakan cara bersusun seperti pada gambar

¹88.58 +146

Kesimpulan

Jadi, hasil pengoperasian bilangan dari 88 + 58 adalah 146 ✓

Penjelasan Soal

Bilangan bulat adalah bilangan yang unsurnya terdiri atas bilangan bulat negatif, bilangan nol, dan bilangan positif.

-a = Negatif0 = nol+a / a = positif

Didalam mengoperasikan campuran bilangan bulat, tentunya kita tahu kata penjumlahan, pengurangan, perkalian, dan pembagian. Apa saja sih itu? perhatian penjelasan berikut ini :

1. Penjumlahan Bilangan Bulat

Penjumlahan adalah operasi aritmatika pertambahan suatu bilangnya yang dilambangkan dengan simbol " + " ( plus ). Sifat-sifat penjumlahan sebagai berikut :

Sifat Komutatif (Pertukaran)

[tex] \boxed{a + b = b + a} [/tex]

Sifat Assosiatif (Pengelompokan)

[tex] \boxed{(a + b) + c = a + (b - c)} [/tex]

Unsur identitas/unsur netral

[tex] \boxed{a + 0 = 0 + a = a} [/tex]

Sifat tertutup

-a + b = c

bilbul + bilbul = bilbul

Jadi semua bilangan bulat jika ditambahkan dengan bilangan bulat hasilnya pasti bilangan bulat juga. Sifat demikian dinamakann sifat tertutup.

Pengoperasian penjumlahan bilbul

a + b = c-a + (-b) = - (a + b)

-a + b = b - a, untuk b > a-a + b = - (a - b), untuk b < a

2. Pengurangan

Pengurangan adalah operasi aritmatika dengan mengurangi suatu bilangan. Pengurangan dilambangkan dengan simbol " - " ( minus ).

Pengoperasian pengurangan bilbul

a - b = a + (-b)-a - (-b) = -a + b

-a - b = - (a + b)a - (-b) = a + b

3. Perkalian

Perkalian adalah operasi penjumlahan secara berulang menurut bilangan yang dikalikannya. Perkalian dilambangkan dengan simbol " × ". Sifat-sifatnya perkalian dalam bilangan bulat sebagai berikut :

Sifat Komutatif :

[tex] \boxed{a \times b = b \times a} [/tex]

Sifat Assosiatif

[tex] \boxed{(a \times b) \times c = a \times (b \times c)} [/tex]

Sifat Distributif

[tex] \boxed{a (b + c) = (a \times b) + (a \times c)} [/tex]

Pengoperasian perkalian bilbul

a × b = c-a × b = -ca × (-b) = -c

Positif × Positif = PositifNegatif × Positif = NegatifPositif × Negatif = Negatif

4. Pembagian

Pembagian adalah operasi pengurangan secara berulang menurut bilangan yang dibaginya. Pembagian dilambangkan dengan simbol " ÷ / : ".

Pengoperasian pembagian bilbul

a ÷ b = c-a ÷ b = -ca ÷ (-b) = -c

Positif ÷ Positif = PositifNegatif ÷ Positif = NegatifPositif ÷ Negatif = Negatif

5. Campuran

Dalam operasi hitung campuran ini, melibatkan antara dua atau lebih operasi hitung bilangan penjumlahan, pengurangan, perkalian, maupun pembagian, bahkan perpangkatan. Dalam operasi hitung campuran, dahulukan operasi sebagai berikut :

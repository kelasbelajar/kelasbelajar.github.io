1. Jelaskan tentang rumus perkalian excel


Untuk menggunakan rumus Perkalian di program Microsoft Excel ada 2 cara, pertama menggunakan operator asterisk (*) dan yang kedua menggunakan formula PRODUCT. Contoh: pada cell A2 masukkan angka 100 dan pada cell B2 masukkan angka 5, kemudian pada cell C2 masukkan rumus = A2*B2 kemudian tekan ENTER hasilnya 500. Sekarang coba rubah rumus pada C2 menjadi =PRODUCT(A2:B2) maka hasilnya akan sama 500

4. Untuk mengawali sebuah rumus di MS Excel digunakan?


Untuk mengawali sebuah rumus di MS Excel digunakan simbol "=" (sama dengan)

PENJELASAN

Microsoft Office Excel adalah aplikasi yang berfungsi untuk mengolah angka. Angka-angka tersebut diolah menggunakan rumus-rumus yang telah disediakan oleh pihak pengembang yaitu Microsoft. Agar dapat menampilkan rumus-rumus tersebut, digunakan simbol "=" (sama dengan).

Berikut beberapa rumus-rumus pada aplikasi Microsoft Office Excel :

=SUM( <sel 1> : <sel 2> ). Rumus ini digunakan untuk menjumlahkan beberapa sel yang berada pada range <sel 1> sampai <sel 2>=AVERAGE( <sel 1> : <sel 2> ). Rumus ini digunakan untuk mencari nilai rata-rata dari beberapa sel yang berada pada range <sel 1> sampai <sel 2>=MAX( <sel 1> : <sel 2> ). Rumus ini digunakan untuk mencari nilai terbesar dari beberapa sel yang berada pada range <sel 1> sampai <sel 2>

6. Yang tau Rumus excel untuk A apa ya?


Jawaban:

Rumus A ialah mencari gaji pokok. Rumusnya dituliskan dalam cell G9. Saya coba dengan Ms. Excel 2016 kak:

=(75%*$B$4)+(0,05*DATEDIF(E9;$B$5;"y")*$B$4)

Penjelasan:

75% berdasarkan ketentuan soal.

$B$4 ialah cell berisi nilai UMP (dijadikan alamat absolut).

0,05 berdasarkan ketentuan soal.

DATEDIF ialah formula untuk mencari selisih waktu (di dalamnya tertulis "y" untuk mengetahui selisih year/ tahun).

28. rumus excel untuk menentukan rata rata


Rumus Ms Excel Menentukan Rata"
=Average(Letak Data Awal:Letak Data Akhir)

Contoh
=Average(E7:E11)Cara Menghitung Rata-Rata Di Microsoft Excel:
1. Posisikan kursor ke baris nilai rata-rata, lalu tuliskan rumus =average(C2:C11), kemudian tekan Enter. Untuk memasukkan rumus secara mudah dan cepat, setelah rumus =average(, tandailah semua data dari kolom C baris 2 sampai baris 11, lalu beri tutup kurung dan Enter.
2. Akhirnya, nilai rata-rata sudah didapatkan.

Bagaimana jika datanya terdiri dari banyak kolom ?

1. Hitunglah nilai rata-rata di kolom paling depan terlebih dahulu, rumusnya ya seperti diatas.
2. Jika sudah, arahkan kursor ke kotak nilai rata-rata kolom pertama sampai muncul tanda plus (+), lalu klik dan tarik sampai kolom paling terakhir. Lepaskan dan rumus secara otomatis akan berlaku pada semua kolom. Jika melakukan seperti ini, kalian tak harus melakukan memasukkan data satu persatu ke kolom excel.

Jawaban:

1. Untuk melakukan penjumlahan dalam Microsoft Excel dapat menggunakan rumus SUM. Rumus SUM ini memiliki fungsi untuk penjumlahan. 2. Berikut adalah rumus yang menggunakan dua rentangsel: =SUM(A2:A4,C2:C3) menjumlahkan angka dalam rentang A2:A4 dan C2:C3. Anda menekan Enter untuk mendapatkan total 39787.

Untuk membuat rumus:

Ketik =SUM dalam sel, diikuti dengan tanda kurung buka (.

Untuk memasukkan rentang rumus pertama, yang disebut argumen (bagian data yang perlu dijalankan rumus), ketik A2:A4 (atau pilih sel A2 dan seret melalui sel A6).

Ketik koma (,) untuk memisahkan argumen pertama dari argumen berikutnya.

Ketikkan argumen kedua, C2:C3 (atau seret untuk memilih sel).

Ketik tanda kurung tutup ), lalu tekan Enter.

Setiap argumen bisa merupakan rentang, angka, atau referensi sel tunggal, semuanya dipisahkan oleh koma.

=SUM(A2:A4,2429,10482)

=SUM(4823,A3:A4,C2:C3)

=SUM(4823,12335,9718,C2:C3)

=SUM(A2,A3,A4,2429,10482)

39. tuliskan rumus penjumlahan, pengurangan, perkalian, dan pembagian pada Microsoft Excel​


Rumus penjumlahan dalam Microsoft Excel dapat menggunakan tombol "+" pada keyboard atau menggunakan rumus SUM dan selalu menggunakan sama dengan (=) di awal penulisan rumus. Salah satu contoh penggunaannya adalah =10+12 atau dengan cell seperti =A1+A2. Di sisi lain, penjumlahan menggunakan rumus SUM dapat dituliskan dengan =SUM(A1:A2).Rumus pengurangan dalam Microsoft Excel dapat menggunakan tombol "-" pada keyboard dan selalu menggunakan sama dengan (=) di awal penulisan rumus. Salah satu contoh penggunaannya adalah =12-10 atau dengan cell seperti =A1-A2.Rumus perkalian dalam Microsoft Excel dapat menggunakan tombol "*" pada keyboard dan selalu menggunakan sama dengan (=) di awal penulisan rumus. Salah satu contoh penggunaannya adalah =10*2 atau dengan cell seperti =A1*A2.Rumus pembagian dalam Microsoft Excel dapat menggunakan tombol "/" pada keyboard dan selalu menggunakan sama dengan (=) di awal penulisan rumus. Salah satu contoh penggunaannya adalah =10/2 atau dengan cell seperti =A1/A2.
